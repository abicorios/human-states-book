Читатель забывает определение через абзац. Исключение: имена собственные людей (но титулы и обращения вроде «саядо», «аджан», «бхиккху», «ринпоче» нужно пояснять).

2. **Замени научные и иностранные слова русскими аналогами:**
   - ментальный → умственный
   - соматический → телесный
   - перцептивный → восприятия
   - когнитивный → умственный
   - эзотерический → тайный
   - доктринальный → теоретический

   - визуализация → мысленное представление
   - концептуальный → понятийный
   - фундаментальный → основной
   - континуум → непрерывный спектр
   - модус → способ
   - компульсия → навязчивое побуждение

3. **Заголовки разделов:** каждый иностранный термин в заголовке должен иметь пояснение в скобках.

4. **Проясняй не только термины, но и смысл.** Если фраза непонятна без контекста (например, «максимум семь перерождений» — кого? куда? зачем?), переформулируй её так, чтобы она читалась самостоятельно.

7. **Аббревиатуры** — расшифровывать при каждом упоминании, не только при первом.

8. **Если перечисляются элементы списка** (например, «из первых шести памятований»), перечисли их кратко в скобках — читатель не помнит, какие именно шесть.

9. **Практические описания:** если упомянут объект практики (касина, мандала, мала и т.д.), кратко поясни, как он выглядит физически.

10. **Используй grep для массовой проверки:** после ручной правки пройди grep по корню каждого иностранного термина и убедись, что ни одно вхождение не осталось без пояснения.
